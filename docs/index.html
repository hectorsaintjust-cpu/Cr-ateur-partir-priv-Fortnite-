Fortnite Private Party Generator

Script pour créer une party privée Fortnite et auto-accepter les demandes de join (amis).

Code de la partie

Si vous créez une partie : copiez le code affiché et utilisez le script local pour créer la party avec ce code. Si vous voulez rejoindre une partie, entrez ce code dans le lobby Fortnite (option "Entrer un code de partie" dans le jeu).

Installer et lancer le script (optionnel)

pip install -U rebootpy
python3 fortnite_private_party_generator.py

Téléchargement du script brut : Télécharger le script

Avertissements

  • Ceci utilise un compte secondaire (alt). Risque de ban si abusé.
  • N'utilisez que pour des lobbys privés avec des amis.

Page statique simple — le bouton "Générer" produit un code localement. Le site ne crée pas la party à votre place ; utilisez le script local ou le jeu pour appliquer le code.
